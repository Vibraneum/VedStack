## üîß TROUBLESHOOTING

**Claude says "I can't see your Airtable":**
- Restart Claude Desktop
- Check config file has Airtable MCP
- Verify PAT in config

**Macros seem wrong:**
- Claude uses reference data from Macro Calculation Rules
- Add more foods to that table for better accuracy
- Provide portion context ("1 katori" vs "large bowl")

**I forgot to log a meal:**
- Just tell Claude: "I ate X at Y time, log it"
- Claude will backdate the entry
